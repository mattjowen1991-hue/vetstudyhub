# VetAce — Veterinary Exam Prep Platform

A web-based study platform to help veterinary students prepare for the NAVLE® and other licensing exams.

## Project Structure

```
vetstudyhub/
├── index.html              # Main landing page
├── css/
│   ├── styles.css          # Core styles, components & layout
│   └── responsive.css      # Mobile/tablet breakpoints
├── js/
│   └── main.js             # Interactivity & animations
├── assets/
│   ├── images/             # Photos, illustrations
│   └── icons/              # SVG icons, favicon
├── .gitignore
└── README.md
```

## Quick Start

Clone and open locally:

```bash
git clone https://github.com/mattjowen1991-hue/vetstudyhub.git
cd vetstudyhub
open index.html
```

No build step required — it's plain HTML/CSS/JS.

## Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, grid, flexbox, animations
- **Vanilla JS** — no framework dependencies
- **Google Fonts** — DM Serif Display + Plus Jakarta Sans
